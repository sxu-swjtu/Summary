> Git在本地有三个区域：工作区、暂存区、本地库
>
> **工作区：**写代码的地方
>
> **暂存区：**将工作区中的操作完成小阶段的存储，是版本库的一部分（有后悔药吃）
>
> **本地库：**存放历史版本，版本库的一部分
